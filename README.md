# Sozdle Project

The parody of the game Wordle.

## Description

Sozdle is a fun and engaging word puzzle game inspired by the popular game Wordle. It aims to provide a similar gameplay experience with some unique twists and features.

## Features

- Interactive word puzzle game
- Color-coded feedback for correct and incorrect guesses
- Backend implemented in Go
- Frontend built with JavaScript, CSS, and HTML
- MVC architecture for clean code organization

## Getting Started

### Prerequisites

- Go (version 1.16 or higher)
- Node.js (version 14 or higher)
- npm (version 6 or higher)

### Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/mmargullan/sozdle-project.git
   cd sozdle-project
   ```

2. Install backend dependencies:
   ```sh
   go mod download
   ```

3. Install frontend dependencies:
   ```sh
   cd frontend
   npm install
   cd ..
   ```

### Running the Application

#### Backend

1. Run the Go server:
   ```sh
   go run main.go
   ```

#### Frontend

1. Navigate to the frontend directory and start the development server:
   ```sh
   cd frontend
   npm start
   ```

2. Open your browser and go to `http://localhost:5173` to see the game in action.
